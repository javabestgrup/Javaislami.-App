JAVAISLAMI — Aplikasi Islam Modern
​JAVAISLAMI adalah aplikasi berbasis web yang dirancang dengan antarmuka modern dan intuitif untuk mendukung aktivitas ibadah harian umat Muslim. Aplikasi ini menggabungkan estetika desain kontemporer dengan fungsionalitas religi yang lengkap, menggunakan palet warna profesional (biru, abu-abu, dan putih) dengan aksen emas yang elegan.
​✨ Fitur Utama
​Aplikasi ini mencakup berbagai modul penting untuk kebutuhan spiritual harian:
​Jadwal Sholat Real-time: Menampilkan waktu sholat dengan indikator sholat berikutnya dan hitung mundur otomatis.
​Al-Qur'an Digital: Daftar surah lengkap dengan tampilan ayat dalam aksara Arab yang indah (font Scheherazade New) dan terjemahan bahasa Indonesia.
​Kompas Kiblat: Fitur navigasi arah kiblat dengan visualisasi kompas yang dinamis.
​Komunitas & Forum: Ruang interaksi bagi pengguna untuk berdiskusi mengenai topik-topik keislaman, marketplace, dan program berbagi (Sedekah/Zakat).
​Edukasi Hukum Islam: Panduan kategori hukum (Wajib, Sunnah, Haram, Mubah) untuk berbagai aktivitas sehari-hari.
​Verse of the Day: Menampilkan kutipan ayat harian untuk inspirasi pengguna.
​🛠️ Detail Teknis
​Teknologi: Dibangun menggunakan HTML5, CSS3 murni (tanpa framework eksternal), dan JavaScript Vanilla.
​Desain Responsif: Dioptimalkan untuk perangkat mobile dengan batas lebar maksimal 420px untuk pengalaman seperti aplikasi asli (Native App Feel).
​Tipografi: Menggunakan kombinasi font Poppins untuk teks modern dan Scheherazade New untuk teks Arab yang autentik.
​UI/UX: Menggunakan sistem navigasi bottom nav bar, animasi transisi halaman, dan skema warna variabel CSS untuk kemudahan kustomisasi
